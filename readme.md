# Generate iOS Assets

* A tool for generating iOS asserts

Example: 

* Put your image in the same folder, name it as "camera 57x57.png". 
* Run the script `python3 GenerateiOSAssets.py`
* Get your images :)
	+ camera.png, 57x57
	+ camera@2x.png, 114x114
	+ camera@3x.png, 171x171
